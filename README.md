# Analyse et Prédiction du Churn Client - Telco

Projet personnel / portfolio Data Science réalisé en 2026.

**Objectif**  
Analyser les facteurs de churn des clients d'une compagnie télécom et construire un modèle de prédiction simple.

**Dataset**  
Telco Customer Churn – Kaggle  
Lien : https://www.kaggle.com/datasets/blastchar/telco-customer-churn  
7043 clients, 21 colonnes (dont Churn = Yes/No)

**Étapes prévues**  
1. Chargement et nettoyage des données (Pandas)  
2. Analyse Exploratoire (EDA) : visualisations avec Seaborn/Matplotlib  
3. Modélisation : Logistic Regression ou Random Forest (Scikit-learn)  
4. Dashboard BI avec Power BI  
5. Conclusions business et déploiement sur GitHub

**Technologies**  
- Python (Pandas, NumPy, Scikit-learn, Seaborn)  
- Power BI  
- GitHub pour versionning
